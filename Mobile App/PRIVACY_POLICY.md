## 1. Introduction

This Privacy Policy explains how **Maheshwari Tech** ("we", "our", or "us") collects, uses, discloses, and safeguards your information when you use the **Apex SMS** mobile application (the "App").

By downloading, installing, or using the App, you agree to the collection and use of information in accordance with this policy.

## 5. Data Storage and Security

### 5.1 Where Data is Stored

All data is stored on **Supabase** cloud servers. Supabase uses secure, encrypted servers (hosted on AWS). Our Supabase project is located at:

- **Supabase Project URL:** `https://aqehjaaikspulflvikcq.supabase.co`

### 5.2 Security Measures

We implement the following security measures:

- **Encryption in transit** — All API communication uses HTTPS/TLS
- **Row Level Security (RLS)** — Database policies restrict data access to authorized users within the same institute
- **Password protection** — User accounts are protected by passwords
- **Session management** — Sessions expire after 24 hours of inactivity
- **Role-based access control** — Different user roles have different data access permissions

### 5.3 Local Storage

The App stores the following data locally on your device using `AsyncStorage`:
- Current user session (encrypted)
- Login timestamp (for session expiry)
