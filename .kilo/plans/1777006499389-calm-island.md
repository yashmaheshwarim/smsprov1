# Plan: Show Enrolled Students When Clicking a Batch in BatchManagementPage

## Context
The current `BatchManagementPage.tsx` displays batches as a grid of cards, each showing batch name, class, status, student count, and subjects. However, clicking on a batch does nothing. The user wants to display the list of enrolled students when a batch is clicked.

## Current State Analysis
- **Batch Display**: Batches are shown as cards in a grid (`grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`), not using DataTable
- **Student Count**: Already displayed on each batch card via `batch.studentCount`
- **No Click Handler**: Batch cards are not clickable currently
- **DataTable Component**: Supports `onRowClick` but we're using cards, so we'll add onClick to card divs

## Implementation Steps

### 1. Add State Variables to BatchManagementPage
Add the following state variables:
- `selectedBatch: Batch | null` — tracks which batch is selected
- `batchStudents: Student[]` — list of students in the selected batch  
- `loadingStudents: boolean` — loading state for student fetch
- `searchStudents: string` — optional search filter for students

### 2. Define Student Interface
Add a local interface for students in a batch:
```typescript
interface BatchStudent {
  id: string;
  name: string;
  enrollment_no: string;
  status: "active" | "inactive" | "graduated";
  created_at: string;
}
```

### 3. Create fetchBatchStudents Function
Add a function to fetch students for a specific batch:
```typescript
const fetchBatchStudents = async (batchId: string) => {
  setLoadingStudents(true);
  try {
    const { data, error } = await supabase
      .from("students")
      .select("id, name, enrollment_no, status, created_at")
      .eq("institute_id", instId)
      .eq("batch_id", batchId)
      .order("name", { ascending: true });
    
    if (error) throw error;
    setBatchStudents(data || []);
  } catch (error: any) {
    toast({ title: "Error", description: error.message, variant: "destructive" });
    setBatchStudents([]);
  } finally {
    setLoadingStudents(false);
  }
};
```

### 4. Modify Batch Card to be Clickable
- Add onClick handler to the batch card div (line 206)
- When clicked: set `selectedBatch` and call `fetchBatchStudents(batch.id)`
- Toggle selection: if same batch clicked again, deselect it
- Add visual indication (border/highlight) for selected batch

### 5. Add Student List Section
Conditionally render below the batches grid when `selectedBatch` is not null:
- **Header**: Show selected batch name + "Back" button to deselect
- **Student DataTable**: Use existing DataTable component with columns:
  - Name (with enrollment no)
  - Enrollment No
  - Status (using StatusBadge)
  - Joined Date (created_at)
- **Loading State**: Show spinner while fetching students
- **Empty State**: Show "No students enrolled in this batch" when empty
