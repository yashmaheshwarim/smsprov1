# Fee Receipt WhatsApp Sharing Implementation Plan

## Overview
Implement functionality to generate fee receipts and share them directly to WhatsApp for specific students in the fee management system.

## Current State Analysis
- FeesPage.tsx contains `generateFeeReceiptPDF` function that creates downloadable HTML receipts
- WhatsApp service exists at src/lib/whatsapp-service.ts but only supports text messages
- Student data includes phone fields: mother_phone, father_phone, student_phone (from migration)
- Zavu service supports file/document sending (messageType: 'document') 
- Fee data is managed through Supabase with tables: student_fees, invoices, batch_fees

## Requirements
1. Generate fee receipts in downloadable format (HTML)
2. Enable direct sharing of receipts to WhatsApp for individual students
3. Maintain existing download functionality
4. Integrate with existing WhatsApp notification system
5. Use student's phone number for WhatsApp sharing

## Implementation Steps

### 1. Enhance Receipt Generation
- Modify `generateFeeReceiptPDF` to optionally return the receipt blob/URL instead of auto-downloading
- Keep existing download behavior as default
- Add parameter to control whether to auto-download or return blob
- Ensure receipt includes all necessary fee details: student info, fees, payments, discounts, etc.

### 2. Extend WhatsApp Service for File Sharing
- Add file/document sharing capability leveraging Zavu API's document messageType
- Create new function: `sendFeeReceiptViaWhatsApp(studentFee, phoneNumber)`
- Use Zavu when available (preferred), fallback to notification with download instructions
- Handle both Zavu (file send) and wa.me (notification) approaches

### 3. Update FeesPage UI
- Add "Share via WhatsApp" button alongside existing "Receipt" button in student fees table
- Button should be disabled when no payment has been made (similar to Receipt button)
- On click: generate receipt blob and initiate WhatsApp share process
- Show appropriate loading/toast messages

### 4. WhatsApp Sharing Logic
**Approach: Direct File Send via Zavu (Preferred)**
1. Generate receipt as Blob (HTML format)
2. Use WhatsApp service to send file as document via Zavu API
3. Fallback to notification approach if Zavu not available or fails

**Fallback Approach: Notification with Download Instructions**
1. Generate receipt and trigger download for user
2. Send WhatsApp message informing user that receipt has been downloaded and can be shared manually
3. Provide guidance on how to share the downloaded file via WhatsApp

```typescript
const generateFeeReceiptBlob = (studentFee: StudentFee, autoDownload = false): Promise<Blob | void> => {
  // Generate HTML content (same as current)
  const receiptContent = generateReceiptHTML(studentFee);
  const blob = new Blob([receiptContent], { type: "text/html" });
  
  if (autoDownload) {
    // Existing download behavior
    const url = URL.createObjectURL(blob);
    const a = document.createElement("a");
    a.href = url;
    a.download = `Fee_Receipt_${studentFee.enrollment_no}_${new Date().toISOString().split('T')[0]}.html`;
    a.click();
    URL.revokeObjectURL(url);
    toast({ title: "Receipt Generated", description: "Fee receipt downloaded successfully." });
    return Promise.resolve();
  } else {
    // Return blob for sharing
    return Promise.resolve(blob);
  }
};
```

```typescript
export const sendFeeReceiptViaWhatsApp = async (
  studentFee: StudentFee,
  phoneNumber: string,
  instituteId: string
): Promise<string> => {
  // Try Zavu first for file sending
  try {
    const zavuSvc = await createZavuServiceForInstitute(instituteId);
    if (zavuSvc) {
      // Generate receipt blob
      const receiptBlob = await generateFeeReceiptBlob(studentFee, false);
      
      // Convert to base64 for sending
      const base64Data = await blobToBase64(receiptBlob);
      
      // Send file via Zavu as document
      const result = await zavuSvc.sendMessage({
        to: phoneNumber.startsWith('+') ? phoneNumber : `+91${phoneNumber.replace(/[^0-9]/g, '')}`,
        messageType: 'document',
        content: {
          filename: `Fee_Receipt_${studentFee.enrollment_no}.html`,
          data: base64Data
        }
      });
      
      // Log success
      await supabase.from('message_logs').insert([{
        institute_id: instituteId,
        channel: 'whatsapp',
        recipient: phoneNumber,
        message: `Fee receipt sent for ${studentFee.student_name}`,
        status: 'sent',
        zavu_message_id: result.message.id,
      }]);
      
      return `zavu:${result.message.id}`;
    }
  } catch (zavuError) {
    console.error('Zavu file send failed:', zavuError);
    // Fall through to fallback
  }
  
  // Fallback: Trigger download and send notification
  await generateFeeReceiptBlob(studentFee, true); // This will auto-download
  
  const message = `Fee receipt for ${studentFee.student_name} (Enrollment: ${studentFee.enrollment_no}) has been downloaded. Please share the downloaded file via WhatsApp manually.`;
  
  // Use existing WhatsApp service for text notification
  return sendWhatsAppAbsentNotification({ 
    phone: phoneNumber, 
    studentName: studentFee.student_name, 
    instituteId, 
    date: new Date().toISOString().split('T')[0] 
  }, message);
};

export const sendWhatsAppAbsentNotification = async (
  notif: WhatsAppNotification,
  customMessage?: string
): Promise<string> => {
  const message = customMessage || `Hello, this is to inform you that ${notif.studentName} is marked ABSENT today (${notif.date}). Please contact the institute for any queries.`;

  // Try Zavu first
  try {
    const zavuSvc = await createZavuServiceForInstitute(notif.instituteId);
    if (zavuSvc) {
      const cleanPhone = notif.phone.replace(/[^0-9+]/g, '');
      const formattedPhone = cleanPhone.startsWith('+') ? cleanPhone : `+91${cleanPhone}`;

      const result = await zavuSvc.sendMessage({
        to: formattedPhone,
        text: message,
        channel: 'whatsapp',
      });

      // Log to Supabase
      await supabase.from('message_logs').insert([{
        institute_id: notif.instituteId,
        channel: 'whatsapp',
        recipient: formattedPhone,
        message: message,
        status: 'sent',
        zavu_message_id: result.message.id,
      }]);

      return `zavu:${result.message.id}`;
    }
  } catch (err) {
    console.error('Zavu WhatsApp send failed, falling back to wa.me:', err);
  }

  // Fallback: Log to database + return wa.me link
  try {
    await supabase.from('message_logs').insert([{
      institute_id: notif.instituteId,
      channel: 'whatsapp',
      recipient: notif.phone,
      message: message,
      status: 'pending',
    }]);
  } catch (e) {
    console.error('Failed to log message:', e);
  }

  const cleanPhone = notif.phone.replace(/[^0-9]/g, '');
  const encodedMsg = encodeURIComponent(message);
  return `https://wa.me/${cleanPhone}?text=${encodedMsg}`;
};
```

#### Helper Function
```typescript
const blobToBase64 = async (blob: Blob): Promise<string> => {
  return new Promise((resolve, reject) => {
    const reader = new FileReader();
    reader.onloadend = () => resolve(reader.result as string);
    reader.onerror = reject;
    reader.readAsDataURL(blob);
  });
};
```

```typescript
const handleShareReceiptViaWhatsApp = async (studentFee: StudentFee) => {
  try {
    // Get student phone number
    const { data: studentData, error } = await supabase
      .from('students')
      .select('student_phone, father_phone, mother_phone')
      .eq('id', studentFee.student_id)
      .single();
    
    if (error) throw error;
    
    // Prefer student phone, fallback to parent phones
    const phoneNumber = studentFee.student_phone || 
                       studentData?.father_phone || 
                       studentData?.mother_phone ||
                       studentData?.mother_phone;
    
    if (!phoneNumber) {
      toast({ title: "Error", description: "No phone number available for this student.", variant: "destructive" });
      return;
    }
    
    // Get institute ID from user context
    const { user } = useAuth();
    const isAdmin = user?.role === "admin";
    const DEFAULT_UUID = "00000000-0000-0000-0000-000000000001";
    const instId = isAdmin ? (user as AdminUser).instituteId : DEFAULT_UUID;
    
    // Share via WhatsApp
    const result = await sendFeeReceiptViaWhatsApp(studentFee, phoneNumber, instId);
    
    if (result.startsWith('zavu:')) {
      toast({ title: "Success", description: "Fee receipt shared via WhatsApp successfully." });
    } else {
      toast({ title: "Info", description: "Receipt downloaded. Please share manually via WhatsApp." });
    }
  } catch (error: any) {
    console.error("Error sharing receipt:", error);
    toast({ title: "Error", description: "Failed to share fee receipt. Please try again.", variant: "destructive" });
  }
};
```
