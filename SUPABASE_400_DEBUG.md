**Likely Causes:** Column mismatch, RLS policy, or data type issues

## MOST LIKELY CAUSES (in order)

1. **Column name typo:** `original_fee` vs `original_fee` (check your SQL file)
2. **RLS blocking:** Disable RLS temporarily to test
3. **Invalid UUID:** `instId` or `student.id` is not a valid UUID format
4. **Foreign key violation:** `batch_fee_id` doesn't exist in `batch_fees` table
5. **Data type mismatch:** Passing string "1000" instead of number 1000.00
